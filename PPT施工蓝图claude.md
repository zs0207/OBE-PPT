# 第二章 OBE导向的人才培养模式创新——PPT施工蓝图 v3.0

## 设计原则

### 1. 按"3次课 × 4学时"组织，不按12个孤立学时

| 次课    | 学时范围   | 主题                                  | 讲义覆盖               |
| ------- | ---------- | ------------------------------------- | ---------------------- |
| **A课** | 第1-4学时  | 订单式培养完整闭环 + 工学交替价值导入 | 2.1全部 + 2.2.1        |
| **B课** | 第5-8学时  | 工学交替实操落地 + 产教融合资源起步   | 2.2.2~2.2.4 + 2.3.1    |
| **C课** | 第9-12学时 | 产教融合系统落地 + 第二章总收束       | 2.3.2~2.3.3 + 总结互动 |

### 2. 开场/过渡页的三级控制

- **课次级开场**：每次课只出现1次（3-4页），说明本次课总目标与结构
- **模块级切换**：2.1→2.2或2.2→2.3的切换，只用1页结构性指引
- **学时级微过渡**：取消独立过渡页；必要的衔接信息放在上一模块最后一页底部或下一模块第一页顶部

### 3. 课间休息处理

不再单独设"课间休息提示页"。在每学时最后一页的底部加注"⏸ 课间休息10分钟"即可。

### 4. 机动页

每次课末尾预留2-3页机动页，用于案例补充/现场互动/节奏缓冲。

### 5. 页数控制

| 次课     | 总页数    |
| -------- | --------- |
| A课      | 82页      |
| B课      | 82页      |
| C课      | 82页      |
| **合计** | **246页** |
